# DVLD - Driving & Vehicle License Department System

A Windows desktop application for managing driving licenses and vehicles, built with C# WinForms following a 3-tier architecture.

## Features
- People management
- Users management with role-based access
- Secure login system with Remember Me option

## Tech Stack
- C# / .NET WinForms
- SQL Server
- ADO.NET
- 3-Tier Architecture

## Architecture
DVLD_WinForm         → Presentation Tier  (UI & Forms)
DVLD_BusinessLayer   → Business Logic Tier (Rules & Validation)
DVLD_DataAccessLayer → Data Access Tier   (Database Operations)
